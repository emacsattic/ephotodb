Show Additional Information

This section provides commands do display additional information and to toggle inline image display on and off.

Show Database

photo-show-database(void); 

Displays the command line used to access the database.

Show Version

photo-show-version(void); 

Displays the Emacs Photo Database version.

Show Manual

photo-show-manual(void); 

Displays the Emacs Photo Database info manual using Emacs' info-mode.

Show Messages

photo-show-messages(void); 

Displays the *photo-messages* buffer which receives the error messages of database clients.

Show Images

photo-show-images(void); 

Displays scans of films, negatives, and prints as inline images in the *photo-output* buffer.

Hide Images

photo-hide-images(void); 

Hide scans of films, negatives, and prints in the *photo-output* buffer and display the paths instead.
